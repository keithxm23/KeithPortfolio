Fantasy Premier League Assistant

August 2013 to Present

A Rails app that scrapes the Fantasy League site and uses a multi-dimensional knapsack to pick the optimal team. Also uses Machine Learning algorithms to predict player performances so as to pick a better team.

Skills

Rails
Machine Learning
jQuery

Wrote a Python script that scrapes the Fantasy Premier League website (http://fantasy.premierleague.com/) for player details and stores it in JSON.

Developed an application in Rails that parses the above JSON data, dumps it to Heroku's Postgres db and then creates a jquery based data-grid (jQGrid) which enables FPL managers to pick better teams and then deployed this application to Heroku

Utilized flotcharts to plot Javascript based charts about interesting information about upcoming players and teams.

Future features include using a multi-dimensional knapsack to pick the best team given a heuristic and then later using Machine Learning to predict player points and pick better teams
